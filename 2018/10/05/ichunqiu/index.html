函数说明：
gzcompress() - 压缩字符串
string gzcompress （string $data [，int $level= -1 [，int $encoding= ZLIB_ENCODING_DEFLATE ]]）
此函数使用ZLIB 数据格式压缩给定的字符串。
参数
data : 要压缩的数据。
level : 压缩程度。可以给出0表示无压缩，最高压缩为9。如果使用-1，则使用zlib库的默认压缩，即6。
encoding : 一个ZLIB_ENCODING_*常数。

gzuncompress - 解压缩压缩字符串
string gzuncompress （string $data [，int $length= 0 ]）
此函数解压缩压缩字符串。
data: 由gzcompress（）压缩的数据。
length: 要解码的最大数据长度。
